# hubot-wwts

What would Todd say? Ask Toddbot! 

See [`src/wwts.js`](src/wwts.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-wwts --save`

Then add **hubot-wwts** to your `external-scripts.json`:

```json
[
  "hubot-wwts"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
